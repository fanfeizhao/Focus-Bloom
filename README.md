# Focus Bloom

Focus Bloom is a Chrome extension that helps users follow structured focus-and-break cycles.

When a focus session ends, the extension opens a fullscreen break window and keeps reopening it if it is closed before the break finishes. This makes breaks harder to skip and encourages users to follow a consistent work-rest routine.

## Features

- Custom focus duration from 10 seconds to 1 hour
- Custom break duration from 10 seconds to 30 minutes
- Sliders and exact minute/second inputs
- Background timer that continues when the extension popup is closed
- Automatic fullscreen break screen
- No skip or cancel button during the break
- Reopens the break window if it is closed early
- Automatically ends the break when the timer reaches zero
- Supports multiple connected displays
- Covers both a computer screen and projector in extended-display mode
- Works automatically with mirrored displays
- Simple On/Off controls
- Turning the timer off immediately resets the current cycle

## Purpose

Many focus timers are easy to ignore, close, or skip. Focus Bloom is designed to make users more committed to their planned focus and break schedule.

The extension does not permanently lock the computer. Instead, it adds enough resistance to discourage users from ending a break early.

## Installation

Focus Bloom is currently installed manually as an unpacked Chrome extension.

1. Download and unzip the project.
2. Open `chrome://extensions`.
3. Turn on **Developer mode**.
4. Click **Load unpacked**.
5. Select the unzipped extension folder.

## Install for Development

After editing the code, return to `chrome://extensions` and click **Reload**.
